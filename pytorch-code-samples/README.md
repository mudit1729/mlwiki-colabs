# PyTorch Code Sample Colab Notebooks

These notebooks are generated from the wiki's Easy, Medium, Hard, and transformer-focused PyTorch implementation pages.

They are PyTorch-only practice workbooks. They intentionally do not include completed answers; each exercise has boilerplate signatures, TODO bodies, and a small `run_exercise_XX_basic_tests()` helper to uncomment after implementation.

## Recommended Hosting

Use GitHub as the source of truth for Colab links:

```text
https://colab.research.google.com/github/mudit1729/mlwiki-colabs/blob/main/pytorch-code-samples/<notebook-name>.ipynb
```

This is easier to maintain than Google Drive links because the notebooks stay versioned and the private wiki can link to a public Colab source.
